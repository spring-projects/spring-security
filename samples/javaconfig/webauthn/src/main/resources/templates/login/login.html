Fast Login means username & password-less login.

Only following combinations are supported for Fast Login as of 2019-06-09.

  • Windows 10 or later, MS Edge, FIDO2 Security Key
  • Windows 10 or later, MS Edge, Windows Hello
  • Windows 10 19H1, Firefox, FIDO2 Security Key
  • Windows 10 19H1, Firefox, Windows Hello
  • MacOS, Chrome Canary, FIDO2 Security Key

If you would like to test two-step authentication with password and security key combination, fill username and password, then press "Login".
